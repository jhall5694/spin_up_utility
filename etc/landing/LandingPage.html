Welcome to TimeClock Plus

Now that you have installed your new Time and Attendance solution you may be wondering, "What's next?" Have no fear, this landing page will direct you to the important applications.

TimeClock Manager

TimeClock Manager is where department heads and time keepers will be doing most of their work. Here you will be able to add new employees, correct time cards, and print reports.

Go to Manager

Admin Server

The Admin Server application allows you to perform administrative functions like updating your software or backing up your database.

Go to Admin Server

Workstation Hub

The Workstation Hub application integrates devices such as a desktop fingerprint reader with TimeClock Plus.

Go to Workstation Hub

WebClock

WebClock offers a complete clock operation and self service solution. Not only will they be able to clock in and out with this application, but also view and approve hours, leave balances, and time off requests.

Go to WebClock

Clock Hub

The Clock Hub application allows users to quickly create a clock hub and edit clocks using that hub. This application provides more detail when viewing or editing clock information.

Go to Clock Hub

Clock Status

Clock Status gives IT and other users in charge of hardware a company-wide overview of which clocks are running and their location. Additionally, terminals that can support multiple companies (such as the RDTg) can be set up here.

Go to Clock Status
